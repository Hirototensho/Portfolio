1  プロジェクト1：留学生の生活実態調査

  • 期間：2022年9〜12月
  • 役割：企画・調査設計・分析を担当
  • 使用ツール：Googleフォーム、Python（集計・可視化）

1.1 背景と目的

　文部科学省「留学生30万人計画」のもとで、2008年以降留学生の受け入れが拡大し、2019年時点で日本国内の高等教育機関では約23万人の留学生を受け入れていた。しかし、2019年末に始まったCOVID-19の感染拡大によって、各国で渡航制限が実施などが実施されたことで2022年には約18万にまで減少しました(白石, 2024)。当時筆者が在学していた私立大学にも多数の留学生が学んでいましたが、留学を中断せざるを得なくなった学生や、日本国内に留まったものの、孤立を経験した学生もいました。
　本プロジェクトは、以上のような背景のもとで、留学生の生活実態を明らかにするべくアンケート調査を実施したものです。

1.2 使用データと手法

　2022年9月から10月初頭にかけて、京都府内の私立大学の学生を対象としてWebアンケート調査を実施しました。回答者の募集は、留学生の多くが利用する学生寮に対するパンフレットの配布に加えて、教員の協力を得て講義などで回答を呼びかける形で実施し、回答は Googleフォームで受け付けました。重複回答を除いた有向回答数は251名で約半数の127名が留学生でした。分析手法として、度数分布表による設問毎の回答者数の集計を行いました。また、アンケート調査と並行して、回答にご協力いただいた留学生を対象として、留学中の生活状況や困りごとに関するヒアリングを実施しました。

1.3 結果の概要

　本調査の主要な結果として、留学生は日本人学生と比べて

  1. 生活満足度が低い（図 1.1 (a)）
  • （留学生の中央値 = 3, 日本人学生の中央値 = 4)
  1. 経済的に困難を感じている学生が多い（図 1.1 (b)）
  2. 住居費が負担になっている学生が多い（図 1.1 (c)）
  3. 困ったときに頼れる人が少ない（図 1.1 (d)）
  • （留学生の中央値 = 3, 日本人学生の中央値 = 5)

という傾向が確認され、留学生は日本人学生と比べて独自の生活上の困難を抱えていることが示唆されました。主要結果1, 2, 4については Wilcoxon の順位和検定で \(p < 0.001\) となり、統計的な有意差があることを確認しています。

　留学生の半数以上が最多支出項目として住宅費をあげた（主要結果3）背景として、アンケート調査と並行して実施したヒアリングにおいて、学生寮を利用できれば住宅費を安く抑えられるものの、受け入れ人数の制約から2年次以上になると寮を出るよう求められることや、留学生が賃貸を借りる上での制度やコミュニケーションのハードルがあることで、住宅費負担が重くなるという証言がありました。
　本調査の結果は学内外の研究報告会にて報告させていただいたほか、ヒアリングの結果と合わせて大学当局に共有し、留学生の支援に役立てていただきました。

1.4 考察

　本調査では、200名以上の学生から回答を集めることができ、学部生が実施するアンケート調査としては成功だっと考えています。その一方で、調査の設計段階でリサーチ・クエスチョンが不明確だったことで設問に不備が残り、十分に説得力のある政策提案に繋げられなかったことなど、反省点の残る調査でもありました。また、2022年秋の一時点における調査という性質上、コロナ禍特有の困難に踏み込めなかったことも悔やまれますし、留学生を取り巻く状況が再び変化した2023年以降の状況を調査することは、今後の課題です。
　以上のようにいくつもの反省点はあったものの、ヒアリングを通じてアンケート調査に協力していただいた学生と直接会う機会もあり、データ分析には身近な人たちの課題を解決できる力があると実感する経験となりました。後に、Chapter 2 で取り上げる「コロナ禍が若年層の雇用に与えた影響」の分析という研究テーマにつながり、大学院に進学するきっかけになったのは間違いありません。
